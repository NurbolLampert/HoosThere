Why I Chose VS Code by Nurbol Lampert (cqq7gs)

I chose VS Code because it has strong support for HTML, CSS, JavaScript, PHP, SQL, and TypeScript, which are essential for this course. Additionally, it provides extensions for Docker and PostgreSQL, making it easier to work with our development environment.

VS Code also integrates well with GitHub and has built-in terminal support, allowing me to work seamlessly without switching between applications.
